


## 📌 Project Overview
This project focuses on analysing data using Microsoft Excel to uncover insights, identify trends, and support data-driven decision-making. The analysis includes data cleaning, transformation, and visualisation.

## 🎯 Objectives
- Clean and prepare raw data for analysis  
- Identify key patterns and trends  
- Create clear and interactive visualisations  
- Provide insights that can support business decisions  

## 📂 Dataset
The dataset used in this project includes:
-Ticket number,priority,type, department etc 

## 🛠️ Tools & Skills Used
- Microsoft Excel  
- Data Cleaning (removing duplicates, handling missing values)  
- Pivot Tables  
- Data Visualisation (charts, dashboards)  
- Basic formulas  

## 🔍 Analysis Process
1. **Data Cleaning**
   - Removed duplicates and errors  
   - Standardised data format  

2. **Data Exploration**
   - Used pivot tables to summarise key metrics  
   - Identified trends and patterns  

3. **Visualisation**
   - Created charts and dashboards  
   - Highlighted key insights visually
